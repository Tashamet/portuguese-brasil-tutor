# Граматика: скорочення усного мовлення (редукції)

Бразильці майже ніколи не говорять так, як пишуть. Щоб розуміти на слух, треба
знати головні скорочення вулиці. Це не «неправильно» — так говорять усі.

## Головні редукції
- **você → cê** ; **você está → cê tá** — «ти / ти є»
- **está → tá** — «є» (*Tá bom* = «добре»)
- **para → pra** — «для/до» (*pra mim* = «мені»)
- **então → tão** (у швидкій мові) — *então tá* → «tão tá»
- **com você → contigo** — «з тобою»
